# CommandBot
An ESP8266 (ESP32) software that controls LEDs or relays through <A href="https://telegram.org/"> Telegram Bot </A> .
You can use <A href="http://espressif.com/en/products/hardware/esp32/overview"> ESP32 ESP-32S </A> Development Board 2.4GHz WiFi+Bluetooth Dual Mode as your mainboard.

You can remotely control things like Leds, relays, garden watering system from your smart phone, tablet, laptop using Telegram application.
